CosmoFish

CosmoFish calculates the Fisher matrices of SN, CMB, WL and galaxy counts and their cross-correlation for a given survey. The original version was written by G.B. Zhao.

CosmoFish is not currently ready for a public release, but a copy of the code can be obtained on request. Please send emails to ahojjati@phas.ubc.ca.
