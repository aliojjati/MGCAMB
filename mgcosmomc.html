MGCosmoMC

Fit Modified Gravity models with CosmoMC

Description

CosmoMC is a Fortran 2008 Markov-Chain Monte-Carlo (MCMC) engine for exploring cosmological parameter space together with Fortran and python code for analysing the sampling. Our MGCAMB patch can be embedded into CosmoMC to fit Modified Gravity (MG) models to data.

The use of our patches is free, however, we request that you cite the following two papers: Testing Gravity with CAMB and CosmoMC and Searching for modified growth patterns with tomographic surveys

Download

The MG patches for the different versions of CosmoMC can be obtained here. The latest version (compatible with CosmoMC July 2015) can be obtained directly from here. Make sure to match the MGCosmoMC version with the CosmoMC version you have to avoid compatibility issues.

Installation

Once you have downloaded the patch, you only need to extract the compressed folder, replace the CosmoMC files with those just extracted and re-compile the code.

Another approach is to manually modify the sources files following these instructions and then recompile the code.

July, 2011: MGCAMB, July-11 version.

This version (V.2) of MGCAMB is written based on CAMB (instead of CAMBsources) which makes it easier to use with Cosmomc. It also generalizes the previous parametrization making it applicable to the entire range of redshifts and includes different components of the energy-momentum tensor. It also includes other modified-gravity parametrizations introduced in the literature.

The previous version of MGCAMB written by G.B. Zhao, L. Pogosian, A. Silvestri and J. Zylberberg was based on CAMBsources. That version modified GR equations for z>30, i.e. at late times when the contribution of radiation is negligible.
